## 코드 블록
---
### 인라인 코드 블록
`#include <stdio.h>`

\`#include <stdio.h>` 
> markdown 문법을 그대로 보여주고 싶을 때는 문법 앞에 `\`를 붙인다.

<br>

### 코드 블록

```java
    public class HelloWorld {
        public static void main(String[] args) {
            System.out.println("Hello World!");
        }
    }
```

\```java
    public class HelloWorld {
        public static void main(String[] args) {
            System.out.println("Hello World!");
        }
    }
```
